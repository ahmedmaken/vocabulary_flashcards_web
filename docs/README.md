# Web Flashcards

This folder contains a static web version of the vocabulary flashcards app.

## Features

- 20 stable-random tests
- All words mode with alphabetical or random order
- Tap-anywhere flashcards
- Saved progress with browser local storage
- Optional word speech with the browser speech API

## Host On GitHub Pages

The simplest option is to put `web-flashcards` in its own GitHub repository and publish from the repository root.

1. Create a new GitHub repository for the web app.
2. Upload the contents of this `web-flashcards` folder to that repository root.
3. Open repository settings.
4. Open Pages.
5. Set the source to `Deploy from a branch`.
6. Choose your main branch and the `/ (root)` folder.
7. Save and wait for GitHub Pages to publish.

The site entry point is `web-flashcards/index.html` in this workspace, or `index.html` if you publish this folder as its own repository.
